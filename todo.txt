# Projects
- Project: Reload  (reload folders of project)
- Project: Save As (save project under a different name while keeping existing project)
- Project: Rename (rename current project)
- Project: Append (add all folders of another project to currently opened)
- Project: Remove folder (remove a folder from currently opened and save)
- OpenFromPrompt
	+ Open [file or folder] from one of the top folders of a project from the list of saved projects.
	+ Ex: Project A
		- Folder 1
			- Folder X
			- Folder Z
		- Folder 2
			- Folder Y
			- Folder W
		- Select Folder 1 or 2, then the sub folder/file to open.
